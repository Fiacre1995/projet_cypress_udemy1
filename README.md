# INSTALATION DE CYPRESS

    1- Initialisation duprojet : **npm init -y**
    2- Install Cypress dans mon projet : **npm install cypress --save-dev**
    3- Exécuté projet cypress : **npx cypress open**

# EXECUTION EN MODE HEADLESS

    1- Execution en mode headless: **npx cypress run**
    Nb : les vidéos sont générés seulement lors de l'exétion en mode headless
